Hide

public struct Hide: AnySideEffect

Navigation action used to ask the Navigator to hide a specific screen identified by the identifierToHide.

The ViewController that is managing that screen must implement RoutableWithConfiguration or Routable in order to be identified with a matching Routable.routeIdentifier.

  • atomic

    Specify if the Hide should generate one single navigation request. For instance, if we have a Route like A/B/C/D and we ask to hide B, with atomic = false, three different Hide commands will be generated: the request to hide D, then the request to hide C and finally the request to hide B. If we use atomic = true, only the request to hide B will be generated.
